Size: 4000 training images, 870 validation images
Bottle Classes: Aquafina, Deer, Kirkland, Nestle
Format: PASCAL VOC, Darknet
Download link: https://drive.google.com/file/d/1D5xOcHndxLHpf5r5d-pzmeli0itoHkuf/view?usp=sharing 
Folder Structure:
 - Annotations: contains the xml label files in PASCAL VOC format
 - ImageSets: contains the training index files 
 - JPEGImages: contains the image data in jpg format
 - Labels: contains the txt label files in Darknet format
